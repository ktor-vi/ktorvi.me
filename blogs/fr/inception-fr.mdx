# Inception : Infrastructure Complète avec Docker

Au **Campus 19 (réseau 42) de Bruxelles**, j'ai développé **Inception**, une infrastructure web complète basée sur des conteneurs Docker.  
Le projet implique l'installation de services tels que Nginx, WordPress, MariaDB, et Redis, orchestrés via Docker Compose.

Composants principaux :
- Reverse proxy (Nginx)
- CMS (WordPress)
- Base de données (MariaDB)
- Cache système (Redis)
- Sécurisation SSL/TLS avec certificats auto-signés

Une plongée passionnante dans la containerisation et l'orchestration système !  
Lien vers le projet : [Inception sur GitHub](https://github.com/ktor-vi/inception).
