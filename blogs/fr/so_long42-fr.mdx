# So Long : Un Petit Jeu 2D

Au **Campus 19 (réseau 42) de Bruxelles**, j'ai développé **So Long**, un petit jeu 2D en utilisant la librairie graphique MiniLibX.  
Le joueur doit se déplacer dans un labyrinthe, récupérer des objets et atteindre la sortie tout en évitant les obstacles.

Fonctionnalités :
- Parsing de cartes en 2D
- Déplacements et collisions du joueur
- Programmation événementielle (clavier)
- Premières animations simples

Une excellente initiation à la programmation graphique en C !
